# 字幕校对

机器转写和翻译难免有错。校对台让你**逐句对照视频**检查和修改字幕，改完直接衔接配音或烧录，不用导出到别的编辑器。

## 打开校对台

三种方式进入：

1. 启动台**工具箱**里点「校对字幕」
2. 转写 / 翻译任务完成后，在最近任务里直接衔接校对
3. 左侧导航「校对」

导入支持三种：**导入视频**（自动检测同目录关联字幕）、**导入字幕**（纯字幕模式）、**导入文件夹**（智能扫描批量建任务）。

- **左侧视频面板**：点任意字幕行，视频跳到对应时间；`Space` 播放 / 暂停
- **右侧字幕列表**：逐条显示时间轴与文本，点击即改，修改自动保存
- **顶部工具栏**：撤销 / 重做、搜索替换、时间轴调整、合并 / 拆分字幕
- **字号三档**（小 / 中 / 大）与面板展开收起，适配不同屏幕

常用快捷键：`↑↓` 上一条 / 下一条，`Tab` 在原文与译文间切换，`⌘S` 保存，`?` 查看全部快捷键。

## AI 帮你改

- **AI 优化**：选中单条让 AI 润色措辞
- **全文 AI 校对**：整份字幕批量送 AI 检查错别字、语病与断句，使用与翻译相同的服务商配置，术语表规则同样生效

## 批量校对任务

多个文件组成一个**校对批次**：列表页显示每个文件的原文字幕 / 翻译字幕匹配状态（自动按文件名匹配，可手动换文件），逐个进入校对，完成度一目了然（`0/3 已完成`）。任务自动保存，随时从启动台的最近任务回开。

- 单条字幕支持**删除并撤销恢复**
- TXT 等不支持的格式会给出明确提示
- 校对完的字幕从原路径直接生效，衔接[配音](./tts-dubbing)或[烧录](./video-merge)
