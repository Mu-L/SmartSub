# 在线视频下载

想给网上的视频做字幕，不用先找第三方下载工具——粘贴链接即可下载，完成后一键进入字幕任务。

## 双下载引擎

| 引擎       | 覆盖                         |
| ---------- | ---------------------------- |
| **yt-dlp** | YouTube 及 1800+ 站点        |
| **lux**    | B 站、抖音、小红书等国内平台 |

首次使用在页面顶部一键安装 / 更新引擎；每条链接可单独选择引擎，也可以让应用按平台自动匹配。

## 使用方式

1. 左侧导航进「下载」，把视频链接粘贴到输入框——**一行一条**，支持从聊天记录等混杂文本中自动识别链接
2. 设置保存目录、清晰度（最佳 / 指定档位）、并发数
3. 「解析链接」预览可用清晰度后下载，或「直接下载」按默认设置开始

下载任务进入最近任务列表，完成后可直接衔接转写 / 翻译任务。

## 官方字幕自动配对

勾选「同时下载官方字幕」后，平台自带的字幕（含自动生成字幕）会随视频一起下载，并在任务向导中**自动配对**——有官方字幕就无需再转写，直接进入翻译或校对，省时且更准确。

## 站点 Cookie：解锁高清与会员内容

不少平台的高清晰度或会员内容需要登录后才能访问。点「站点 Cookie」导入你的登录态，三种方式：

1. **浏览器一键提取**：从本机浏览器自动读取目标站点 Cookie
2. **cookies.txt 文件**：导入浏览器扩展导出的标准格式文件
3. **手动粘贴**：直接粘贴 Cookie 字符串

:::caution
Cookie 等同于你的登录凭证，仅保存在本地并只用于下载请求；请勿把含 Cookie 的配置分享给他人。
:::
