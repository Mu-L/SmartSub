# 字幕翻译

把生成或导入的字幕翻译成目标语言。可以在任务向导里与转写串成流水线，也可以单独用「翻译已有字幕」批量处理现成的 SRT 文件。

## 基本流程

1. 启动台选「翻译已有字幕」（或在任何任务里勾选「翻译字幕」目标）
2. 选择**源语言**、**目标语言**与**翻译服务**
3. 选择输出内容：**纯译文**，或**原文 + 译文**双语字幕
4. 开始批量翻译

翻译服务下拉按配置状态分组，已配置的排在前面。任一服务未配置时，去「翻译」页面完成配置即可（见下文）。

## AI 翻译的质量保障

AI（大模型）翻译质量高，但模型行为需要约束，妙幕内置了多层防护：

- **对齐防护**：自动识别并修复模型合并 / 拆分字幕造成的译文错位，确保译文与原文一一对应；可开启「回显对齐校验」让模型对每条字幕同时返回原文回显与译文，系统据此检测修复
- **结构化输出**：测试连接时自动探测服务商支持的结构化输出模式并写回配置，增强返回格式的可靠性
- **失败重试**：只重试失败的批次，不浪费已成功的结果
- **思考模式开关**：默认关闭思考让翻译更快更省；需要深度推理时可随时打开（自动按服务商下发对应参数）

## 术语表：专有名词翻译更统一

在「词库」页面维护术语表（品牌名、人名、专业词），翻译时命中的词条会自动注入提示词，多词库支持优先级排序与 CSV / TXT 导入导出。

## 批量与性能

- **批量翻译数量 / 批次并发数 / 请求间隔**三个参数直接在服务配置页并排调节，大批量字幕翻译速度可数倍提升
- 每个 AI 服务可配置**自定义请求参数**（String / Float / Boolean / Array / Object / Integer），界面直接添加修改、实时校验，支持导出导入，无需改代码
- AI 翻译结果受模型和提示词影响较大，可在[提示词设置](../advanced/custom-prompts)中调整模板

## 输出形式

- **纯译文**：只保留翻译结果
- **原文 + 译文**：双语字幕（上下两行），适合语言学习与对照检查
- 自定义翻译后的字幕文件名后缀，方便播放器识别语言轨
