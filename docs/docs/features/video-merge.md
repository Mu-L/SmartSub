# 视频合成（字幕烧录）

把字幕（和可选的配音音轨）合成进视频，得到可以直接发布的成品。编辑器是剪辑软件风格：左侧调样式，中间实时预览，底部时间轴轨道。

## 两种合成方式

| 方式                  | 说明                                                 | 适合                     |
| --------------------- | ---------------------------------------------------- | ------------------------ |
| **烧录硬字幕**        | 字幕永久渲染进画面，任何播放器 / 平台都能显示        | 发布到视频平台、投屏播放 |
| **封装软字幕（MKV）** | 以流复制方式无损封装可切换字幕轨，速度极快、画质无损 | 本地收藏、多语言切换     |

## 字幕样式：所见即所得

- **预设样式**：经典白字黑边、电影字幕、YouTube 风格、黑底白字、清新简约、醒目加粗
- **我的样式**：调好的样式一键「存为我的样式」，跨项目复用
- **精细控制**：字体、字号、字体颜色、描边模式 / 背景框模式、九宫格位置
- 中间画面**实时渲染预览**，拖动时间轴任意查看

## 配音音轨

在顶部第三个文件位放入[配音工作台](./tts-dubbing)导出的音轨，可与字幕一起合成，直接产出「翻译字幕 + 外语配音」的成品视频。

## 画质与编码

- **导出画质**：原画质 / 高 / 标准档位
- **编码方式**：CPU（体积更小）或**硬件加速**（速度快数倍；相同画质下体积可能比 CPU 编码大 30%–100%，对体积敏感时用 CPU）
- 硬件加速自动探测可用编码器（macOS VideoToolbox、Windows NVENC / QSV），不可用时禁用并说明原因；合成中途失败会自动回退 CPU 重跑

## 操作步骤

1. 启动台工具箱点「合成到视频」，或从任务完成处衔接
2. 选择视频与字幕（可加配音音轨），预览确认样式
3. 选择烧录硬字幕或封装软字幕、画质与编码方式
4. 确认输出路径，点「生成视频」
