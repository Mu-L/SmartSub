# 翻译服务选型总览

翻译服务负责把字幕翻成目标语言。**免费翻译开箱即用**；AI 翻译质量更高、支持术语与上下文，需要 API Key；传统机翻稳定快速、按量计费。任一就绪即可在任务中选用。

## 三类服务对比

| 类别         | 服务                                                                                                                                                                                       | 免费情况             | 质量特点                         |
| ------------ | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ | -------------------- | -------------------------------- |
| **免费起步** | [内置免费翻译](./free)、[Ollama 本地模型](./ollama)、[DeepLX](./deeplx)、[谷歌翻译](./google)                                                                                              | 免费为主             | 日常够用，零或低成本             |
| **AI 翻译**  | [DeepSeek](./deepseek)、[Gemini](./gemini)、[通义千问](./qwen)、[硅基流动](./siliconflow)、[DeerAPI](./deerapi)、[Azure OpenAI](./azure-openai)、[自定义 OpenAI 兼容](./openai-compatible) | 部分有免费额度       | 语义准确、可调提示词、支持术语表 |
| **传统机翻** | [百度](./baidu)、[阿里云](./aliyun)、[火山](./volcengine)、[豆包](./doubao)、[小牛](./niutrans)、[腾讯](./tencent)、[讯飞](./xunfei)、[微软](./azure)                                      | 多家有月度免费字符量 | 稳定快速、按量计费，适合大批量   |

## 按需求推荐

1. **零配置起步**：「自动免费翻译」聚合多个免费源、自动回退，开箱即用——先用它跑通流程
2. **质量优先**：配 [DeepSeek](./deepseek) 或 [Gemini](./gemini) 的 API Key，质量明显更好（Gemini 的 Key 可免费创建）
3. **完全离线**：[Ollama](./ollama) 跑本地大模型，数据不出本机
4. **大批量成本控制**：传统机翻按字符计费、多家有免费额度（百度 / 火山 / 腾讯等），限流参数可调
5. **已有中转 / 自建服务**：[自定义 OpenAI 兼容](./openai-compatible)接入任意端点

## AI 翻译的通用能力

所有 AI（大模型）服务共享这些机制，配置页直接可调：

- **批量翻译数量 / 批次并发数 / 请求间隔**：控制吞吐与限流
- **回显对齐校验**：让模型返回原文回显，自动检测修复译文错位（token 消耗约增一倍）
- **思考模式**：默认关闭（更快更省），需要深度推理时打开
- **高级选项**：[系统 / 用户提示词模板](/advanced/custom-prompts)；**配置参数**：自定义请求参数
- **[术语表](/advanced/glossary)**：命中词条自动注入
- **测试翻译**：保存前先试跑，自动探测结构化输出模式

## 配置入口

左侧导航「翻译」页面：侧栏选服务商 → 填凭据 → 「测试翻译」→ 就绪。服务商按配置状态分组显示，已配置的排前面。
