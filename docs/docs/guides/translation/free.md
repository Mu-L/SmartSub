不想注册任何服务？装好妙幕就自带三个免费翻译源：

| 服务                     | 说明                                                     |
| ------------------------ | -------------------------------------------------------- |
| **自动免费翻译**（推荐） | 聚合多个免费源，**一个失效自动切换下一个**，内置限速保护 |
| 必应免费翻译             | 单独使用必应的免费接口                                   |
| 谷歌免费翻译             | 单独使用谷歌的免费接口（部分网络环境需代理）             |

## 在妙幕中配置

默认已就绪，无需任何配置。想微调时在「翻译」页面选中对应服务：

<div className="img-container">
  <img src="/img/v3/translation/auto-free.webp" alt="自动免费翻译配置：回退链与每分钟最大请求数" />
</div>

- **回退链**（自动免费翻译）：源的尝试顺序，失败自动切换
- **每分钟最大请求数**：限速保护，避免被接口封禁；批量任务偶发失败时调低它

## 使用建议

- 免费接口为公共资源，**质量与稳定性不做保证**：正式发布内容建议用 [AI 翻译](./overview#按需求推荐)复核或[校对台](/features/proofreading)过一遍
- 免费源不支持[术语表](/advanced/glossary)注入（那是 AI 翻译的能力）
- 大批量高频使用请自觉控制速率，或改用按量计费的正规服务
