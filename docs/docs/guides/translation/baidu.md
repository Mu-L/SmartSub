## 申请步骤

1. 注册 [百度翻译开放平台](https://fanyi-api.baidu.com/)
2. 进入「产品服务 → 通用文本翻译」开通服务，**建议完成个人认证**升级高级版（免费额度 5 万 → 100 万字符/月，QPS 也更高）
3. 创建应用后，在「开发者信息」页面获取 **APP ID** 与**密钥**

## 在妙幕中配置

「翻译」页面选「百度翻译」：

| 字段         | 填写                                      |
| ------------ | ----------------------------------------- |
| APP ID       | 开发者信息中的 APP ID                     |
| Secret Key   | 开发者信息中的密钥                        |
| 批量翻译数量 | 每次翻译的句子数，**最大 18**，过大会失败 |

点「**测试翻译**」验证后即可使用。

## 常见问题

- **54003 访问频率受限**：标准版 QPS 只有 1——完成认证升级高级版，或把请求间隔加大到 1 秒以上
- **52003 未授权**：APP ID / 密钥不匹配，或服务未开通
- **术语定制**：在百度控制台的术语库维护（应用内[词库](/advanced/glossary)只作用于 AI 翻译）
