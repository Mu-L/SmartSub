字节跳动的豆包语音合成大模型：中文音色自然度出众（如「爽快思思 2.0」等大模型音色），并支持**声音复刻 2.0** 高保真克隆。

## 申请步骤

1. 注册 [火山引擎](https://www.volcengine.com/) 并实名认证
2. 进入**豆包语音控制台**，在「开通管理」开通**语音合成大模型**
3. 在「**API Key 管理**」签发 API Key（与[豆包听写](/guides/cloud-asr/volcengine)同一控制台，可复用）

## 在妙幕中配置

「音色」页面 → 在线服务选「豆包语音」：

<div className="img-container">
  <img src="/img/v3/tts/doubao.webp" alt="火山引擎豆包语音配置表单" />

填入 API Key，配置音色候选，点「**测试连接**」验证后即可在配音工作台使用。

## 声音复刻 2.0

用自己的声音做高保真克隆（效果优于本地 ZipVoice，但需付费）：

1. 在火山控制台**购买声音复刻的音色槽位**
2. 在妙幕「音色」页点「添加音色」→ 选「**火山复刻 2.0**」
3. 按向导提供参考音频（或现场录音）→ 克隆完成后音色进入「我的音色」
4. 配音工作台里选用该音色，云端合成

:::tip 云端音色可找回
云端克隆的音色在本地误删后，可从平台重新取回，不会丢失槽位。
:::

## 常见问题

- **测试失败**：确认「开通管理」里已开通语音合成大模型；方舟平台的 Key 不通用
- **复刻效果**：参考音频质量决定上限——安静环境、单人清晰、10 秒以上
- **同账号还能做什么**：[豆包听写](/guides/cloud-asr/volcengine)、[火山翻译](/guides/translation/volcengine)、[豆包翻译](/guides/translation/doubao)
