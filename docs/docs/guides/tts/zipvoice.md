零样本声音克隆模型（约 217 MB / 24kHz，中 / 英）：**一段参考音频 + 对应文本，即建即用**，无需训练等待。克隆出的音色保存在「我的音色」，可离线反复使用。

## 第一步：下载模型

「音色」页面 → 本地模型 → 「ZipVoice 声音克隆」→ 下载：

## 第二步：创建克隆音色

点「去创建克隆音色」（或侧栏「添加音色」）进入四步向导：**选素材 → 选片段 → 参考文本 → 命名保存**。

<div className="img-container">
  <img src="/img/v3/tts/clone-wizard.webp" alt="创建克隆音色向导：三种克隆方式与素材选择" />
</div>

- **素材来源**：选择音频 / 视频文件（mp3、wav、m4a、mp4、mkv 等），或**直接用麦克风录一段**
- **自动选片段**：系统会自动截取素材中质量最好的片段（建议素材 10 秒以上）
- **质量检测**：创建时自动检测时长、信噪比、削波、音量并给出修复建议，内置降噪

### 什么样的素材效果好

- 安静环境录制，无背景音乐和噪音
- 单人清晰说话，避免多人对话
- 语气自然平稳，贴近你想要的配音风格

## 第三步：使用与管理

- 克隆完成后音色出现在「**我的音色**」：可试听样本、播放参考音频、重新生成、重命名
- 在[配音工作台](/features/tts-dubbing)选择「我的音色」即可整片配音
- 支持**导出 / 导入音色**（跨设备迁移）；音色详情页保留创建时的质量检测结果
