科大讯飞的「录音文件转写大模型」。与其它同步服务不同，它是**异步订单制**：上传后服务端排队转写，**退出应用也不丢任务**，下次打开继续取结果——特别适合超长录音。

## 申请步骤

1. 注册 [讯飞开放平台](https://www.xfyun.cn/) 并实名认证
2. 创建应用，开通**录音文件转写大模型**服务
3. 在应用的「**服务接口认证信息**」中获取 **APPID / APIKey / APISecret** 三件套

## 在妙幕中配置

「引擎」页面 → 云端听写分组选「讯飞听写」：

<div className="img-container">
  <img src="/img/v3/cloud-asr/xfyun.webp" alt="讯飞录音文件转写配置表单：APPID、APIKey、APISecret 与语种档位" />
</div>

| 字段               | 填写                                                                          |
| ------------------ | ----------------------------------------------------------------------------- |
| APPID              | 应用的数字 ID                                                                 |
| APIKey / APISecret | 服务接口认证信息中的密钥对                                                    |
| 模型               | 语种档位：`autodialect`（中文与方言自动，默认）或 `autominor`（含更多小语种） |

点「**测试连接**」验证后即可使用。

## 异步订单制说明

- 转写任务提交后由讯飞服务端处理，妙幕轮询取结果；**中途退出应用不影响**，重启后自动续查
- 单次上传超时默认放宽到 300 秒（大文件上传耗时较长），一般无需调整

## 常见问题

- **识别语言怎么选**：语种免切自动识别，任务里的「原语言」仅做上传前的支持性检查
- **超长音频**：应用会自动压缩并按限制切片（约 48MB / 段），无需手动分割
- **同账号还能做什么**：[讯飞翻译](/guides/translation/xunfei)
