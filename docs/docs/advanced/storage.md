# 存储位置与缓存

语音模型动辄几百 MB 到几个 GB。**统一存储目录**让你在一个地方管理所有模型与临时文件的存放位置，不再逐项修改。

## 统一存储目录

「设置 → 存储位置」：

- **统一存储目录**：whisper 模型、faster-whisper 模型、FunASR 等 sherpa-onnx 模型、本地 TTS 模型、运行时与临时文件都收纳在这个目录下，一处设置全部生效
- 想把模型放到大容量磁盘 / 外接硬盘？点「浏览」换个目录即可

:::caution 更改目录不会自动搬家
修改只改变**读写位置**：旧目录中已下载的模型不会自动迁移，可手动移动到新目录的同名子目录，或重新下载。
:::

- **打开目录**：直达当前存储目录，方便手动导入模型文件
- **清除**：恢复为默认位置（系统应用数据目录）

## 临时文件目录

转写与合成过程会产生音频切片等临时文件。默认放在统一存储目录下的 `temp/`；也可以打开「使用自定义临时文件目录」单独指定（优先于统一存储目录）。

- **清除缓存**：一键清理临时文件
- 自定义临时目录不会被自动清理策略误删文件，请手动定期清理

## 相关设置

同页还有几个常用项：

- **有执行任务时阻止系统休眠**：长任务不再因息屏 / 休眠中断（默认开）
- **网络代理**：为下载、更新检测和翻译服务统一设置代理
- **下载源（高级）**：模型与组件的下载源选择（国内加速 / 官方直连）
