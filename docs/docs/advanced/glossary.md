# 术语表（词库）

品牌名被翻错、人名前后不一致，是 AI 批量翻译最常见的痛点。词库让你**一次定义、处处生效**：命中的术语自动注入提示词，翻译任务、校对重译和翻译 AI 优化使用同一规则。

## 使用方式

1. 左侧导航进「词库」，新建一个词库（如「品牌与产品词」）
2. 新增词条：**原文 → 期望译文**，备注可选（备注也会提供给模型作翻译提示）
3. 打开词库开关。之后所有 AI 翻译任务自动生效，命中词条会写入日志方便核查

## 多词库与优先级

- 可以建多个词库（比如「品牌词」「人名」「行业术语」），按列表顺序**从上到下优先**
- 多个启用词库出现相同原文时，采用列表中最靠上的词条
- 每个 AI/LLM 批次只注入**当前源字幕实际命中**的词条，不浪费 token

## 批量导入导出

- 支持 **CSV / TXT** 文件批量导入，页面提供 CSV 模板下载
- 模板列为 `source, target, note`（备注列可选）；需要完整保留字段内容时推荐 CSV
- 词库可导出分享给同事，跨设备同步

## 适用范围

:::info 只作用于 AI 翻译
应用内词库注入提示词的机制只对 AI / 大模型翻译生效。百度、阿里、火山、腾讯等**传统机翻**的术语请在对应厂商控制台维护（词库页面提供各家控制台的直达链接）。
:::
