# 任务配方

每次都手动选一遍「目标 + 模型 + 语言 + 翻译服务」很浪费时间。**配方**把任务向导里的整套配置存下来，下次一键复用。

## 保存配方

1. 在任务向导里配置好一个任务（我要得到什么、语音模型、源语言 / 目标语言、翻译服务、高级设置……）
2. 点底部的「**存为配方**」，命名保存（如「英文视频 → 中英双语」）

## 使用配方

- 保存的配方以卡片形式出现在**启动台**的开始创作区域，与内置任务卡并列
- 点配方卡 → 拖入文件 → 开始，配置零调整
- 文件也可以**直接拖到配方卡上**立即开任务

## 管理配方

- 配方支持**重命名**与删除
- 配置有变化时，重新走一遍向导另存新配方即可

## 适合的场景

- 周更栏目：每期视频固定「转写 + 翻译 + 烧录」同一套参数
- 多语言分发：为每个目标语言存一个配方（英→日、英→西……）
- 团队协作：统一约定命名与参数，减少口头交代
