# 没有显卡也能用：云端听写方案

轻薄本、老电脑跑不动本地大模型？**云端听写**把转写交给在线服务：不占本机资源、速度与电脑性能无关，也不用下载几个 GB 的模型——填一个 API Key 就开工。

:::tip 免费额度就够用
腾讯云每月**免费 5 小时**、Gladia 每月**免费 10 小时**、Groq 免费档也很慷慨。轻度用户可以常年零成本。
:::

## 第一步：选服务商并拿到 Key

| 你在意的                | 推荐                                         | 免费情况                             |
| ----------------------- | -------------------------------------------- | ------------------------------------ |
| 中文内容 + 稳定免费额度 | [腾讯云](/guides/cloud-asr/tencent)          | 每月 5 小时                          |
| 额度更大 / 小语种       | [Gladia](/guides/cloud-asr/gladia)           | 每月 10 小时（免费档音频会用于训练） |
| 注册最快                | [Groq](/guides/cloud-asr/groq)               | 免费档限速可用                       |
| 中文最强（愿付费）      | [火山豆包听写](/guides/cloud-asr/volcengine) | 按量计费                             |

点开对应指南照着注册，几分钟拿到密钥。

## 第二步：在妙幕里配置

「引擎」页面 → 左栏「云端听写」分组 → 选中服务商 → 填入凭据 → 点「**测试连接**」：

<div className="img-container">
  <img src="/img/v3/engines/overview.webp" alt="引擎总览：云端听写就绪状态" />
</div>

看到绿色「可用」就绪。可以同时配置多家，任务里随用随切。

## 第三步：正常跑字幕任务

启动台选任务 → 拖入音视频 → 「语音模型」下拉里选择你配置的云端服务 → 开始。转写期间音频会自动压缩 / 切片上传，长音频也不怕。

:::caution 隐私提醒
音频会上传到你配置的服务商端点处理（首次使用有确认提示）。**敏感内容请不要走云端**——需要保密时换本地轻量模型（`tiny` / `base` 在 CPU 上也能跑）。
:::

## 搭配建议

- **翻译也免费**：内置免费翻译零配置，或 [Gemini 免费 Key](/guides/translation/gemini)
- **偶尔要离线**：备一个 `base` 本地模型（148 MB）应急，CPU 可跑
- **超长录音**：选[讯飞听写](/guides/cloud-asr/xfyun)——异步订单制，关掉应用任务也不丢
