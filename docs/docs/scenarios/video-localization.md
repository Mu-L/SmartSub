# 视频出海：翻译字幕并配外语音轨

把中文视频做成英语（或任何语言）版本：字幕翻译 → 外语配音 → 合成导出，一条龙在妙幕里完成。还可以用**克隆的自己声音**说外语，保持人设一致。

:::tip 省事版
启动台直接选「**视频 → 配音成片**」任务卡：听写、翻译、配音、合成一条龙全自动出成品。本页拆开讲每一步，便于精细控制质量。
:::

## 第一步：得到外语字幕

启动台「视频 → 双语字幕」，源语言中文、目标语言英语：

- 翻译服务建议用 AI 档（[DeepSeek](/guides/translation/deepseek) / [Gemini](/guides/translation/gemini)），语感自然
- 建好[术语表](/advanced/glossary)统一品牌与人名译法
- 完成后进[校对台](/features/proofreading)过一遍——**配音之前把文字定稿**，省得返工

## 第二步：外语配音

把译好的字幕（+ 原视频）送进[配音工作台](/features/tts-dubbing)：

<div className="img-container">
  <img src="/img/v3/dubbing/workbench.webp" alt="配音工作台：选择引擎音色，逐行合成与试听" />
</div>

1. **选声音**：
   - 用自己的声音说外语：[ZipVoice 克隆](/guides/tts/zipvoice)（免费）或 [ElevenLabs IVC](/guides/tts/elevenlabs)（多语效果佳）
   - 现成音色：[Azure](/guides/tts/azure)（700+ 音色）/ [ElevenLabs](/guides/tts/elevenlabs) / 本地 [Kokoro](/guides/tts/local-engines)（免费）
2. **开始配音**：逐条合成并自动对齐时间轴；语速预控 + 静音间隙借用让配音贴轴
3. **处理超长行**：进入人工清单的行——精简译文文案（最有效）、单行重生成或接受变速
4. **输出形态**：选「替换音轨的视频」或「混音（压低原轨）」，也可只导出音轨给剪辑软件

## 第三步：合成发布版

到[视频合成](/features/video-merge)出成品：

- 放入视频 + 外语字幕（+ 配音音轨），选一个字幕样式预设
- **烧录硬字幕**（平台通用）或**封装软字幕 MKV**（观众可开关）
- 编码方式选「硬件加速」提速数倍（对体积敏感用 CPU）

## 多语言批量策略

- 每个目标语言[存一个配方](/advanced/recipes)（英 / 日 / 西 ……），后续视频一键复用
- 翻译层批量跑，配音层按语言分批处理
- 时间轴以「配音后导出的对齐字幕」为准，避免字幕与语音错位

## 免费路径

| 环节 | 免费方案                                     |
| ---- | -------------------------------------------- |
| 翻译 | 内置免费翻译（质量要求高换 Gemini 免费 Key） |
| 配音 | 本地 Kokoro（含英语音色）/ ZipVoice 克隆     |
| 合成 | 内置 ffmpeg                                  |
