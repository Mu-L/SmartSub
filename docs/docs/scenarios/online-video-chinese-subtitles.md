# YouTube / B 站视频怎么加中文字幕

看到一条外语视频想配中文字幕？妙幕内置视频下载 + 转写 + 翻译，粘贴链接到出中文字幕，全程一个应用搞定。

:::tip 这条流程可以完全免费
本地模型转写 + 内置免费翻译，不需要任何 API Key。视频若自带官方字幕还能跳过转写，更快。
:::

## 第一步：粘贴链接下载视频

左侧导航进「**下载**」：

<div className="img-container">
  <img src="/img/v3/download/video-download.webp" alt="视频下载页：粘贴链接、选择引擎与清晰度" />
</div>

1. 把视频链接粘贴进输入框（一行一条，支持批量；从聊天记录复制的混杂文本也能自动识别）
2. 引擎按平台自动匹配（YouTube 走 yt-dlp，B 站走 lux），首次使用先在页面顶部安装引擎
3. **勾选「同时下载官方字幕」**——视频若带官方字幕（含自动字幕），后面可以直接翻译、跳过转写
4. 选清晰度点「解析链接」或「直接下载」

需要高清 / 会员内容时导入[站点 Cookie](/features/video-download#站点-cookie解锁高清与会员内容)。

## 第二步：生成 / 获取原文字幕

下载完成后在最近任务点击衔接，或回启动台把视频拖到「**视频 → 双语字幕**」卡片上：

- **有官方字幕**：任务向导会自动配对字幕文件，直接进入翻译
- **无官方字幕**：选择语音模型（英文内容推荐 `large-v3-turbo` 或轻量的 `base.en`）、源语言选「英语」，开始转写

没有下载模型？在[引擎页](/guides/engines/overview)一键下载，或改用[云端听写](/guides/cloud-asr/overview)。

## 第三步：翻译成中文

任务向导里勾选「翻译字幕」目标：

- **零成本**：翻译服务选「自动免费翻译」（默认可用，无需配置）
- **质量更好**：配一个 [DeepSeek](/guides/translation/deepseek)（几块钱翻几十集）或 [Gemini（免费 Key）](/guides/translation/gemini)
- 输出选「**原文 + 译文**」得到双语字幕，或纯译文

点「开始」，完成后字幕文件（`.srt`）生成在视频同目录。

## 第四步：使用字幕

- **播放器直接挂载**：PotPlayer / IINA / VLC 等会自动识别同名字幕文件
- **压进视频**：想发布或投屏，就用[视频合成](/features/video-merge)烧录硬字幕
- **先校对再用**：机器翻译拿不准的地方进[校对台](/features/proofreading)逐句核对

## 常见问题

- **下载失败**：确认链接有效；会员内容需要 Cookie；引擎可在下载页检查更新
- **翻译人名 / 术语不统一**：建一个[术语表](/advanced/glossary)统一译名
- **批量处理一整个播放列表**：把多条链接一次粘贴进下载框即可
