## ADDED Requirements

### Requirement: 提供面向搜索意图的场景教程

文档站 SHALL 提供「场景教程」分类，首批 MUST 覆盖四个场景：给 YouTube/B 站在线视频生成中文字幕（含视频下载功能）、播客/会议录音批量转文字、视频出海配外语音轨（翻译 + TTS 配音 + 合成）、无显卡电脑使用云端听写转写。每篇教程 SHALL 以端到端步骤组织（准备 → 操作 → 产出），并链接涉及的功能篇与服务商配置页。

#### Scenario: 场景教程端到端可跟随

- **WHEN** 用户按「视频出海配外语音轨」教程操作
- **THEN** 教程覆盖从导入字幕/视频、选择翻译服务、配音合成到导出成品的完整步骤，每步链接到对应详细文档

#### Scenario: 场景标题即搜索词

- **WHEN** 查看任一场景教程的页面 title 与 H1
- **THEN** 标题直接使用用户搜索表述（如「YouTube 视频怎么加中文字幕」），而非内部功能名

### Requirement: 场景教程标注免费路径

每篇场景教程 SHALL 明确标注该场景是否可完全免费完成，并给出免费方案的具体组合（引擎/服务选择）。

#### Scenario: 免费路径清晰可见

- **WHEN** 用户浏览「播客录音转文字」教程
- **THEN** 页面明确给出零成本组合（本地模型 + 内置免费翻译），无需申请任何 API Key
